# Astronomy-Picture-of-the-Day
# 🚀 NASA APOD Explorer

Aplicación web interactiva que consume la API de la NASA para mostrar la *Astronomy Picture of the Day (APOD)*, permitiendo explorar imágenes astronómicas por fecha y guardar favoritas en el navegador.

---

## Descripción

Este proyecto permite a los usuarios:

- Visualizar la imagen o video astronómico del día
- Consultar APODs de fechas pasadas
- Leer la explicación científica de cada publicación
- Guardar sus APODs favoritas
- Acceder rápidamente a sus favoritos almacenados

---

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- API pública de la NASA
- LocalStorage (almacenamiento en navegador)

---

## API utilizada

- Endpoint: https://api.nasa.gov/planetary/apod
- Documentación oficial: https://api.nasa.gov/

---

## Funcionalidades

### 1. Foto del día
- Obtiene automáticamente la APOD actual desde la API
- Muestra:
  - Título
  - Fecha
  - Imagen o video
  - Explicación

### 2. Búsqueda por fecha
- Permite seleccionar una fecha pasada
- Validación de fechas futuras

### 3. Sistema de favoritos
- Guardar APODs en favoritos
- Persistencia usando localStorage
- Lista de favoritos interactiva
- Eliminación de favoritos
- Carga automática al hacer clic
